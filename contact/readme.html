Contact - a modern contact module!

This module is a fork of a older xoops contact module. Big part of the code is rewritten and more functions are added.

It uses bootstrap templates which makes it perfect for modern websites

  • Features
    • Bootstrap templates
    • Option to use Google reCaptcha
    • Embed Google Map to your contact form
    • Contact Block only with the form, or only with the map or both
    • Send Confirmation Mail to sender.
    • Collect ICQ - on or off
    • Collect URL - on or off
    • Collect Skype - on or off
    • Collect Company - on or off
    • Collect Location - on or off
    • Collect Phone - on or off
    • Collect Address - on or off
    • Send Mails to different departments
    • Set Contact Email
